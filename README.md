# Personal Portfolio Website

## Project Overview
This is a simple personal portfolio website built using HTML5.

## Features
- About Section
- Skills Section
- Contact Form
- Navigation Menu
- Profile Image
- Semantic HTML Tags

## Technologies Used
- HTML5

## How to Run
1. Download project files.
2. Open index.html in any browser.
3. Navigate through sections using menu links.
